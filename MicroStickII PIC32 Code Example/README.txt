MicroStick 2nd Gen BlinkLED Code Example

--------------------------------------------------
Code Example Description:
--------------------------------------------------
BlinkLED.mcp is a simple example demonstrating how to access the LED function.

This example will blink the D6 LED on the Microstick II at an increasing speed until it appears ON and then repeats.
